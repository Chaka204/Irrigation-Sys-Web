# AgriLink Smart Irrigation System

Real-time monitoring and control web interface for AgriLink smart irrigation system.

## Features
- Real-time sensor data display
- Remote pump control
- Automatic/manual mode switching
- Alarm notifications
- Activity logging

## Setup
1. Update `app.js` with your Firebase configuration
2. Push to GitHub
3. Enable GitHub Pages

## Live Demo
Visit: [your-username.github.io/agrilink-web](https://your-username.github.io/agrilink-web)

## Hardware
- ESP8266 Microcontroller
- Soil Moisture Sensor
- DHT11 Temperature/Humidity Sensor
- Ultrasonic Water Level Sensor
- Relay Module
- LCD Display
